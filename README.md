# Neural Networks
This is test development of some basic neural networks.


## 🧐 Features    
- Solve simple problems like XOR
- Working on low level with no external modules
        

## 🧑🏻‍💻 Usage
```bash
node neuron-<type>.js
```
        

## 🛠️ Tech Stack
- [Node.js](https://nodejs.org/)
